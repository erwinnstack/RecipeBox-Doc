# RecipeBox-Doc

## Descriere
Acesta este un proiect pentru o aplicație simplă de rețete. Scopul aplicației este de a oferi o listă de rețete și detalii complete pentru fiecare rețetă, inclusiv ingrediente și instrucțiuni pas cu pas.

### Structura dosarului
- **Home**: Pagina principală care conține lista rețetelor disponibile.
- **content,content_1,content_2,Adaug**: Pagina care afișează detaliile fiecărei rețete, inclusiv ingredientele și instrucțiunile de preparare sau adauga retete.

### Funcționalități
1. **Pagina principală**:
   - Afișează o listă cu rețete, fiecare rețetă având un link către pagina sa detaliată.
2. **Pagina detaliată a rețetei**:
   - Afișează ingredientele necesare, poza și pașii pentru prepararea rețetei selectate.
3. **Pagina de adaugare a unei rețete**:
   - Afișează un meniu unde adaugăm denumirea, ingredientele, instrucțiunile și poza.
